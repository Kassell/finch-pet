# Pet

住在桌面上的 Finch 桌宠。悬浮在所有窗口之上,不挡状态栏和 Dock,也不出现在调度中心;它能感知 Agent 的运行状态——后台会话完成时会冒气泡提醒你,双击气泡直接打开对应会话。

## 玩法

- **拖拽**:随手拖到任意位置,支持多屏;快速甩出去会带惯性滑行。
- **单击** 打招呼,**双击** 蹦一下;**右键** 让它回家(隐藏)。
- 对话里直接吩咐即可:"换只宠物"、"让宠物说加油"、"把宠物藏起来"。

## 换宠物

内置奶茶水獭 `boba`。兼容 [Petdex](https://petdex.dev) 宠物包,把 Petdex 页面链接、图片 URL、本地图片 / 文件夹 / zip 包发给 Agent 说"添加宠物"就能导入(spritesheet 按 8 列 × 9 行网格解析,基准帧 192×208,支持同网格的高清图)。
